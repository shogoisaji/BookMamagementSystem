# Book Management System

### ツール

- Laravel
- tailwindcss
- Breeze
- supabase
- lottie

### 機能

- 新規登録
- ログイン
- 在庫書籍一覧表示
- 書籍詳細確認
- コメント、おすすめ度の入力
- 書籍のレンタル、返却
- 書籍の登録（admin only）
- 書籍登録用の書籍検索 Google Books Api
- アカウント管理
